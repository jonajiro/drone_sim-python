# drone_sim-python
pythonのドローンシミュレーター</br>
ゲームパッドのアナログスティックで操作可能</br>
無人航空機操縦士実地試験の８の字飛行練習用</br>
drone_sim.py実行で起動</br>
以下、ゲームパッド機能割り当て</br>
左スティック上下：スロットル</br>
左スティック左右：方位角</br>
右スティック上下：前進後進</br>
右スティック左右：左進右進</br>
STARTボタン：状態リセット</br>
R上ボタン：軌跡非表示</br>
L上ボタン：軌跡表示</br>
